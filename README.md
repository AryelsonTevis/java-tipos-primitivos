# Tipos Primitivos em Java

Projeto desenvolvido durante meus estudos de Java para praticar os conceitos fundamentais da linguagem.

## Conceitos abordados

* Tipos primitivos (`int`, `float`, `double`, `char`, `boolean`)
* Conversão de tipos (casting)
* Entrada e saída de dados
* Operações básicas

## Tecnologias utilizadas

* Java
* NetBeans

## Como executar

1. Clone o repositório.
2. Abra o projeto no NetBeans.
3. Execute a classe principal.

## Objetivo

Consolidar os conhecimentos iniciais da linguagem Java e compreender o funcionamento dos tipos de dados primitivos.
